Documentation de l'API

  • GET /

    • Documentation de l'API


  • GET /demandes

    • Liste de toutes les demandes en attente de vote Une redirection vers la demande sera présente pour permettre le vote


  • GET /demandes/{index}

    • Affichage de la demande correspondante à l'index si elle existe


  • POST /demande

    • URL appelée lors de la création d'une demande avec les paramètres suivants :

    • type : Type de la demande choisie parmis les différents type proposés
    • label : Sujet de la demande
    • origin : Dépositaire de la demande
    • Réponse OUI :
      • href : URL de redirection
      • accept : VRAI
      • label : Nom du choix
    • Réponse NON :
      • href : URL de redirection
      • accept : FAUX
      • label : Nom du choix


  • PUT /demande/{index}

    • Lien de modification de la demande index avec les données modifiées (voir format d'une demande)


  • POST /demande/{index}

    • URL de vote d'une demande. Redirection vers l'URL OUI ou NON
